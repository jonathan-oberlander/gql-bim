# gql-yoga with codegen and biome set-up

Yoga
Fastify
Logger
GQL Codegen
Linter & formatter Biome
Bundler
    - tsx check for simple type check
    - tsc for dev
    - esbuild for prod

dev: ```npm run dev```
visit [localhost](http://localhost:4000/graphql)
build: ```npm run build```
start: ```PORT=8080 npm start```
